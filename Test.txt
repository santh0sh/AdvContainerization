Slide: Jenkins Quiz - Question

What is a Jenkins pipeline used for?

A) Managing databases

B) Automating deployment and testing

C) Hosting websites

D) Monitoring performance



---

Slide: Jenkins Quiz - Answer & Explanation

Answer: B) Automating deployment and testing

Explanation:
A Jenkins pipeline is used to define a series of automated steps to build, test, and deploy applications, making the CI/CD process more streamlined and efficient.

Slide: Jenkins Quiz - Question

Which scripting language is commonly used for writing Jenkins pipelines?

A) Python

B) Groovy

C) JavaScript

D) Ruby



---

Slide: Jenkins Quiz - Answer & Explanation

Answer: B) Groovy

Explanation:
Jenkins pipelines are typically written using the Groovy scripting language, which allows for flexibility and control over build processes.
